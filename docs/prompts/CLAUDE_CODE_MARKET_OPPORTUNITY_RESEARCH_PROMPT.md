# Claude Code — Market Opportunity Research Prompt (Type B, research-only)

Template for a **research-only** prompt about a discovered market opportunity.
Generated by
`portfolio_automation.claude_code_prompts.generate_market_opportunity_research_prompt(card)`.

This is research evidence ONLY. It never becomes an official recommendation and
never produces a buy/sell order. Private candidates (e.g. SpaceX) are evaluated
for **access routes only** (IPO watch / public suppliers / ETFs / proxies) — they
are not directly tradeable.

Every generated prompt contains: candidate context + scores, research questions,
expected output (a research note recommending at most "send to sandbox / watchlist
review"), and the **Forbidden block** (no trading / orders / broker writes / money
movement / allocation changes).
